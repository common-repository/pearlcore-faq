Pearlcore FAQ is fully responsive with Unlimited Categories, Custom Styling and Like features. You Can also add FAQ's on WooCommerce Product.

== Description ==

<h4>FAQ content</h4>

<p>
Manage frequently asked questions on your site with the help of Pearlcore FAQ plugin. 
Pearlcore FAQ Plugin comes with a full featured administration dashboard where you can manage question and questions categories. 
On the configuration section you can easily style your FAQ's.
</p>

<p>
Allow User to ask question on product and other FAQ's. Setting for show/hide Ask Question form. 
Shortcode to display form with Faq's [Pc_Faq ask_question="true"]. 
Shortcode to display Category Based Faq's [Pc_Faq category="sport"].
</p>

<h4>Add FAQ On WooCommerce Products</h4>
<p>
On Products Section of WooCommerce you can easily manage FAQ's for every single product. You can Add and Delete Question. On Configuration panel
you can enable/disable WooCommerce Product FAQ feature for all products or manage this feature on each product. 
</p>

== Frequently Asked Questions ==

= How To show category based FAQ's ? =

Place this shortcode with your desired category name. [Pc_Faq category="category_name"]

= Ask Question form is not showing under FAQ's ? =

Place this shortcode to display Ask Question form. [Pc_Faq ask_question="true"]

= Title is not showing ? =

Place this shortcode to display title. [Pc_Faq title="Frequently Asked Questions"]

== Installation ==

1. Unzip downloaded  `pearlcore-faq.zip` file.
2. Place pearlcore-faq folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add shortcode [Pc_Faq] or place widget in sidebar.
5. You are done.
